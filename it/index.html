Use Signal Horse as your local trading workspace for AI agents

Signal Horse è uno spazio di lavoro di trading locale e un'API HTTP per agenti IA e operatori. Permette di connettere Binance, OKX, Bybit, Bitget, Gate e altri exchange supportati con un clic per il trading automatizzato dall'IA, il trading manuale e il batch trading su più account. Chart, saldi, posizioni ed esecuzione ordini restano su un'unica macchina; le credenziali rimangono in locale. OpenClaw è un client supportato, ma il flusso non è limitato ad esso.

Cosa viene installato sulla tua macchina

Signal Horse viene installato sulla stessa macchina su cui fai trading. L'installazione scarica il programma, crea file di dati locali e può abilitare l'avvio automatico affinché il tuo strumento IA si connetta sempre allo stesso servizio locale.

Programma principale Signal Horse

È il programma locale installato sul tuo computer. Fornisce la pagina di trading, la connessione locale e le azioni di trading.

Impostazioni e registrazioni locali

Le tue impostazioni e altri dati locali rimangono su questa macchina instead of in un account web remoto.

Avvio automatico in background

Dopo l'installazione, Signal Horse può avviarsi automaticamente in background, senza doverlo riaprire manualmente ogni volta.

Exchange key entry

Per connettere un exchange, compila API Key e Secret. OKX e Bitget richiedono anche una passphrase.

Dettagli di installazione

Se installi autonomamente, usa l'installer standard. Per preparare un server o un workflow IA, usa il comando di installazione in una riga.

Windows

Portable ZIP for the local browser-based trading UI. No admin rights, registry writes, or auto-start.

Download Windows portable ZIP Windows x64 · 1.0.0+20260511.053243Z

Extract the ZIP into any user-writable folder, then run the local launcher:

1. Download and extract signal_horse_windows_portable.zip
2. Open the extracted folder
3. Run Start Signal Executor.cmd
4. Your browser opens the local UI

If you prefer a scripted install into LocalAppData, use the PowerShell helper:

Invoke-WebRequest -Uri https://signal.horse/install.ps1 -OutFile install.ps1
powershell -ExecutionPolicy Bypass -File .\install.ps1 -Yes

The local UI opens at:

http://127.0.0.1:38182/

This Windows package stays in user space. It does not require admin rights, does not write startup registry entries, and only notifies about new versions.

macOS

App per la barra dei menu che installa Signal Horse localmente e permette di aprire la pagina di trading o riavviare il servizio locale con un clic.

Download Signal Horse for macOS (.dmg) Scarica DMG Intel macOS 13+ · Apple Silicon / Intel

Per la maggior parte degli utenti macOS, usa l'app pacchettizzata:

1. Scarica e apri il pacchetto macOS Signal Horse (.dmg)
2. Trascina l'app scaricata in /Applications
3. Apri l'app e usa l'icona nella barra dei menu per aprire l'interfaccia locale nel browser
4. Usa il menu per riavviare il servizio locale o attivare l'avvio in background

Questo DMG può essere aperto normalmente nel Finder, ma non è ancora notarizzato da Apple. La prima volta, macOS potrebbe chiederti di fare clic destro sull'app e scegliere Apri una volta.

Per deployment da riga di comando o installazione automatica, puoi sempre usare il comando di installazione in una riga:

curl -fsSL https://signal.horse/install.sh | bash -s -- --non-interactive --skip-credentials --install-service

L'installazione da riga di comando configura lo stesso servizio locale Signal Horse. L'app è semplicemente un modo più comodo per gestirlo dal desktop.

Linux (Ubuntu / Debian)

Esegui un comando e Linux scaricherà Signal Horse, lo installerà, configurerà l'avvio automatico e aprirà il servizio locale sulla porta 38182:

curl -fsSL https://signal.horse/install.sh | bash -s -- --non-interactive --skip-credentials --install-service

Al termine, conferma che Signal Horse sia in esecuzione:

curl http://127.0.0.1:38182/health

Per aprire la pagina Signal Horse da un altro dispositivo, consenti la porta TCP 38182 nel firewall Linux o nel gruppo di sicurezza, poi visita:

http://<server-ip>:38182/

Per impostazione predefinita, Signal Horse usa la porta 38182 su Linux. L'installer cerca di configurare l'avvio automatico per l'utente corrente; senza root, viene installato nella directory utente.

FAQ

Quick answers to the questions most users ask before they start.

Questo installerà Signal Horse sul mio computer?

Sì. L'installer scarica Signal Horse sul tuo computer e può mantenerlo in esecuzione localmente in background per il tuo strumento IA o la pagina di trading.

Le mie chiavi API exchange passano attraverso il tuo server?

No. Le tue chiavi API rimangono sulla tua macchina. Signal Horse le legge localmente e invia le richieste direttamente all'exchange.

Which port does Signal Horse use?

Per impostazione predefinita, Signal Horse usa la porta 38182. Sulla stessa macchina, apri http://127.0.0.1:38182/ . Per accedervi da un altro dispositivo Linux, apri prima quella porta nel firewall.

Come restare al sicuro abilitando il trading IA?

Inizia con la modalità demo, mantieni la conferma manuale per gli ordini reali e imposta limiti di leva e rischio prima di far piazzare ordini reali all'IA.
