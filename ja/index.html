エージェント統合 · ローカルエグゼキューター

Use Signal Horse as your local trading workspace for AI agents

Signal HorseはAIエージェントとオペレーター向けのローカル取引ワークスペースおよびHTTP APIです。Binance、OKX、Bybit、Bitget、Gateなど対応取引所にワンクリックで接続し、AI自動取引・手動取引・複数口座のバッチ取引をサポートします。チャート、残高、ポジション、注文実行はすべてローカルマシンに留まり、取引所の認証情報もローカルに保管されます。OpenClawは対応クライアントの一つですが、使用方法はOpenClawに限定されません。

マシンにインストールされるもの

Signal Horseは取引を行うマシンと同じマシンにインストールされます。セットアップはプログラムをダウンロードし、ローカルデータファイルを作成し、自動起動を有効にすることでAIツールが常に同じローカルサービスに接続できます。

Signal Horseコアプログラム

コンピューターにインストールされるローカルプログラムです。取引ページ、ローカル接続、取引アクションを提供します。

ローカル設定と記録

設定やその他のローカルデータはリモートWebアカウントではなく、このマシンに保存されます。

自動バックグラウンド起動

インストール後、Signal Horseはバックグラウンドで自動的に起動するため、毎回手動で開く必要はありません。

Exchange key entry

取引所を接続するには、APIキーとSecretを入力してください。OKXとBitgetはパスフレーズも必要です。

インストールの詳細

自身でインストールする場合は通常のインストーラーを使用してください。サーバーやAIワークフローを準備する場合は、1行インストールコマンドを使用してください。

Windows

Portable ZIP for the local browser-based trading UI. No admin rights, registry writes, or auto-start.

Download Windows portable ZIP Windows x64 · 1.0.0+20260521.021641Z

Extract the ZIP into any user-writable folder, then run the local launcher:

1. Download and extract signal_horse_windows_portable.zip
2. Open the extracted folder
3. Run Start Signal Executor.cmd
4. Your browser opens the local UI

If you prefer a scripted install into LocalAppData, use the PowerShell helper:

Invoke-WebRequest -Uri https://signal.horse/install.ps1 -OutFile install.ps1
powershell -ExecutionPolicy Bypass -File .\install.ps1 -Yes

The local UI opens at:

http://127.0.0.1:38182/

This Windows package stays in user space. It does not require admin rights, does not write startup registry entries, and only notifies about new versions.

macOS

Signal Horseをローカルにインストールし、取引ページを開いたりローカルサービスをワンクリックで再起動できるメニューバーアプリです。

Download Signal Horse for macOS (.dmg) Intel DMGをダウンロード macOS 13+ · Apple Silicon / Intel

ほとんどのmacOSデスクトップユーザーには、パッケージ済みアプリを使用してください：

1. Signal Horse macOSパッケージ（.dmg）をダウンロードして開く
2. ダウンロードしたアプリを/Applicationsにドラッグする
3. アプリを開き、メニューバーアイコンを使用してブラウザでローカルUIを開く
4. メニューを使用してローカルサービスを再起動またはバックグラウンド起動をオンにする

このDMGはFinderで通常通り開けますが、まだApple公証を受けていません。初回はmacOSがアプリを右クリックして「開く」を選択するよう求める場合があります。

コマンドライン導入や無人セットアップが必要な場合は、1行インストールコマンドを引き続き使用できます：

curl -fsSL https://signal.horse/install.sh | bash -s -- --non-interactive --skip-credentials --install-service

コマンドラインインストールは同じローカルSignal Horseサービスをセットアップします。アプリは単純にデスクトップからより便利に管理する方法です。

Linux（Ubuntu / Debian）

1つのコマンドを実行すると、Linuxはオーガニックにorsignal Horseをダウンロードし、インストールし、自動起動を設定し、ポート38182でローカルサービスを開きます：

curl -fsSL https://signal.horse/install.sh | bash -s -- --non-interactive --skip-credentials --install-service

完了後、Signal Horseが動作していることを確認してください：

curl http://127.0.0.1:38182/health

別のデバイスからSignal Horseページを開く場合は、LinuxファイアウォールまたはセキュリティグループでTCPポート38182を許可してから、以下にアクセスしてください：

http://<server-ip>:38182/

デフォルトでは、Signal HorseはLinuxでポート38182を使用します。インストーラーは現在のユーザーの自動起動を設定しようとし、rootを使用しない場合はユーザーディレクトリにインストールされます。

推奨初回起動フロー

最も簡単な初回起動：Signal Horseをインストールし、取引所のAPIキーを追加し、残高とポジションを確認してから、デモまたはライブ取引を選択します。

1
Signal Horseをローカルにインストールする

AIツールまたは取引ワークフローが動作する同じマシンにインストールしてください。

2
Add exchange API keys

APIキーとSecretを入力してください。OKXとBitgetはパスフレーズも必要です。

3
取引前にデータを確認する

ライブ注文を出す前に、残高、ポジション、注文履歴が正しく読み取れることを確認してください。

4
実行モードを選択する

まずデモまたはテストネットモードを使用してください。アカウントと権限が正しいことを確認した後のみライブ取引に切り替えてください。

よくある質問

Quick answers to the questions most users ask before they start.

Signal Horseがコンピューターにインストールされますか？

はい。インストーラーはSignal HorseをコンピューターにダウンロードしAIツールや取引ページのためにバックグラウンドでローカルに動作し続けます。

取引所のAPIキーはあなたのサーバーを経由しますか？

いいえ。APIキーはご自身のマシンに留まります。Signal Horseはローカルで読み取り、リクエストをマシンから取引所に直接送信します。

Which port does Signal Horse use?

デフォルトでは、Signal Horseはポート38182を使用します。同じマシンでhttp://127.0.0.1:38182/を開いてください。別のLinuxデバイスからアクセスする場合は、まずファイアウォールでそのポートを開いてください。

AI取引を有効にする際の安全対策は？

デモモードから始め、実注文には手動確認を維持し、AIにライブ取引を許可する前にレバレッジとリスク制限を設定してください。

Supported software platforms

Signal HorseはOpenClaw、Claude Code、Codex、Hermes Agentsなどのエージェントプラットフォームから接続でき、複数の暗号通貨取引所への一元アクセスと統合管理を実現します。

OpenClaw

ローカルエージェントワークフローは取引所アクセス、注文実行、取引自動化のためにSignal Horseを呼び出せます。

Claude Code

Claude Codeをリサーチ、手動実行、自動化ワークフローのためのローカル取引コパイロットとして使用してください。

Codex

CodexはSignal Horseを暗号資産リサーチ、注文実行、AI取引自動化のためのローカル取引ゲートウェイとして使用できます。

Hermes Agents

Hermes AgentsなどのマルチエージェントランタイムはSignal Horseを通じて暗号通貨取引所の操作を統合できます。
